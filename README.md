# Full Project Idea
- Our project will be an automated AI enumeration tool designed to analyze system and network reconnaissance data. The tool will accept outputs from tools such as network scans and local enumeration utilities and will use an AI-driven decision engine to correlate findings and identify relevant security insights. We will use Kali Linux machines for testing, write all project code in Python, and utilize an API for AI integration.
